Detailed Description

Static thread-safe console logger with configurable severity filtering.

Features:

  • Thread-safe: synchronized via internal static std::mutex.
  • Supports 4 severity levels (DBG, INFO, WARN, ERROR).
  • Global filter level configurable at runtime (set_min_level()).
  • Each line includes timestamp + level + message.

Usage example:

Logger::set_min_level(Logger::Level::INFO);
Logger::info("Worker started");
Logger::warn("Task queue is almost full");
Logger::error("Unhandled exception in worker");

void Logger::log ( Level  lvl,
const std::string &  msg 
)
static

Core logging function that prints a message with explicit severity.

Core method that handles message formatting and output.

Parameters
lvlSeverity level (DBG, INFO, WARN, ERROR).
msgThe message to print.

All wrapper methods (debug(), info(), etc.) call this internally. This function checks the minLevel, then prints a timestamped message.

Note
Thread-safe: a global std::mutex prevents output interleaving.
Parameters
lvlSeverity level of the message.
msgMessage text to print.

GIVEN a message and a severity level, WHEN log() is invoked, THEN:

  • If lvl < minLevel, the message is ignored.
  • Otherwise, the message is printed to std::cout as:
    [YYYY-MM-DD HH:MM:SS] [LEVEL] message
Note
  • Thread-safe: all access to std::cout is serialized with a mutex.
  • Uses std::endl to flush output immediately.

Sets the minimum log level.

Sets the minimum severity level to be printed.

Parameters
lvlThe minimum level to display.

GIVEN a desired log verbosity, WHEN set_min_level() is called, THEN only messages with level >= lvl will be printed to stdout.

Example:

Logger::set_min_level(Logger::Level::WARN);
Logger::info("This will be ignored");
Logger::error("This will be shown");
Parameters
lvlThe minimum log level; messages below this level are ignored.

GIVEN a running application, WHEN set_min_level() is called, THEN subsequent calls to log() will only print messages with level >= lvl.

Thread-safe: uses a lock to protect shared configuration.
